# Симуляция автономного гусеничного робота в Unity

## Описание проекта

Данный проект представляет собой прототип автономного гусеничного робота, разработанный в среде Unity.

В рамках проекта была создана виртуальная среда с препятствиями и целевым объектом. Робот способен перемещаться как в ручном режиме с помощью клавиатуры, так и в автоматическом режиме, выполняя движение к заданной цели.

## Реализованный функционал

* Создание 3D-сцены в Unity.
* Импорт и настройка модели гусеничного робота.
* Ручное управление роботом с помощью клавиш W, A, S, D.
* Автоматическое наведение на целевой объект.
* Автоматическое движение к цели.
* Остановка робота при достижении цели.
* Реализация датчика на основе Raycast.
* Подготовка системы обнаружения препятствий.

## Структура проекта

### RobotController

Скрипт ручного управления роботом.

### TargetFollower

Скрипт автоматического движения к цели.

### SimpleSensors

Скрипт датчика, реализованного с использованием Raycast.

### Simulation Environment

Тестовая сцена, содержащая робота, препятствия и целевой объект.

## Используемые технологии

* Unity 6
* C#
* Raycast
* 3D Simulation

## Перспективы развития

В дальнейшем планируется расширение проекта за счёт:

* системы объезда препятствий;
* нескольких датчиков окружающей среды;
* применения методов обучения с подкреплением (Reinforcement Learning);
* интеграции Unity ML-Agents;
* взаимодействия с реальной робототехнической платформой.

## Результат

В результате был создан рабочий прототип автономного гусеничного робота в виртуальной среде Unity, способный обнаруживать цель и перемещаться к ней в автоматическом режиме.
